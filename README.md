# My Portfolio

A personal portfolio website for Ifa Bogale, built with HTML, CSS, and JavaScript. The site highlights skills, services, projects, experience, education, and contact information with a responsive layout and theme toggle.

## 🔗 Live Sites

- Vercel: <https://my-portfolio1-com.vercel.app>
- GitHub Pages: [My Portfolio](https://ifabogg.github.io/my-portfolio1.com/)

## 🚀 Features

- Responsive layout for desktop and mobile
- Theme toggle with local storage persistence
- Mobile navigation menu
- Skills, experience, education tabs
- Portfolio and project showcase sections
- Contact form submission using Google Apps Script
- Scroll-to-top button

## 🧰 Built With

- HTML
- CSS
- JavaScript
- Font Awesome icons
- Google Apps Script for contact form submissions

## 📁 Project Structure

- `index.html` - main page content and layout
- `style.css` - visual styling and responsive design
- `script.js` - interactive behavior and UI scripts
- `pdf/` - resume file download
- `photo/` - portfolio images and profile photo

## 💡 Notes

- The contact form is wired to a Google Apps Script endpoint for form submission.
- The project is deployed on Vercel and GitHub Pages for live preview.
